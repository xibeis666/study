gitHub笔记
git init 初始化一个库
git status 查看库状态
git add 文件名,把一个文件加入到暂存区里
git add . 把当前目录所有文件加入到暂存区中
git commit -m '这里写注释'
git log 查看版本
git checkout 版本id(commit 后的 171a5f6c55162a2a04592393bfcebb4e2a148eaa)
git checkout -- . 回到上一个版本
git checkout -b 新分支名 复制原有分支，开一个新的分支
git branch 查看分支
git checkout 分支名 切换分支
git merge 分支名 -m '注释信息 合并分支
git clone 克隆项目
首次提交 :
git remote add origin https://github.com/xibeis666/study.git git push -u origin master
